    API(공공 데이터)
    데이터 크롤링 -> 추출 기술
        크롤링: 기어다니듯이 탐색 지속(랜덤으로 돌아다님)
        스크래핑: 문서를 분석해 특정 패턴을 지난 데이터를 추출하는 순서로 진행
                추출한 데이터를 원하는 형태로 사용하게 db에 저장(페이지를 지정)

    서버 사이드 렌더링: 서버에서 렌더링 해 클라이언트에게 전달(정적, jsoup)
    클라이언트 사이드 렌더링: 웹 페이지 초기 로딩 레이아웃과 자바 스크립트 코드를 전달하고
        브라우저가 자바 스크립트를 실행해 데이터를 불러와 동적으로 웹 페이지 생성(동적, selenium)
        검색이 안 돼 SEO(검색 엔진 최적화)로 검색 기능을 추가
